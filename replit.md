## Project Description (from README)
SmartSearch is described as an intelligent chatbot and voice assistant designed to help users find available benefits, payments, social support measures, and useful information. The service is intended to work in **Russian** and **Yakut languages** with the following features:

### Planned Features
1. **Speech-to-Text Recognition**
   - Russian language support
   - Yakut language support
   - Mixed speech (code-mixing)

2. **Bilingual Support**
   - Russian
   - Yakut
   - Mixed form

3. **Benefits and Payments Search**
   - Social benefits and payments
   - Regional programs
   - Pension benefits
   - Disability support
   - Housing subsidies
   - Multi-child family benefits
   - Student and youth benefits

4. **Text-to-Speech Response**
   - Voice output
   - Adaptive speech speed and style
   - User-friendly for elderly users

### Architecture Notes
The README mentions:
- Yandex Translator API for Yakut-to-Russian translation
- Voice input/output capabilities
- Search functionality for social benefits
