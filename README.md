# SmartSearch

**SmartSearch** — это интеллектуальный чат-бот и голосовой ассистент, который помогает пользователям находить доступные льготы, выплаты, социальные меры поддержки и полезную информацию.  
Сервис работает на **русском** и **якутском языке**, поддерживает **голосовой ввод**, **голосовые ответы**, а также умеет **переводить речь с якутского на русский через API Яндекс Переводчика**.

---

## Возможности

---

### 1. Распознавание речи (Speech-to-Text)

Ассистент умеет понимать голосовые запросы:
- русский язык  
- якутский язык  
- смешанная речь (код-мешинг)

---

### 2. Поддержка двух языков

Ассистент принимает запросы:
- на русском  
- на якутском  
- в смешанной форме  

---

### 3. Поиск льгот и выплат

Может находить:
- льготы для льготников  
- социальные выплаты  
- региональные программы  
- пенсионные льготы  
- поддержку инвалидам  
- субсидии на ЖКХ  
- льготы многодетным семьям  
- студенческие и молодёжные льготы
- ...  

Ассистент объясняет:
- кому положено  
- какие документы нужны  
- куда обращаться  
- сроки и условия  
- требования по доходам  
- механизм подачи
- ...

---

### 4. Голосовой ответ (Text-to-Speech)

Помимо текстового ответа, ассистент:
- произносит ответ голосом  
- адаптирует скорость и стиль речи  
- делает ответы понятными для пожилых пользователей
